
# 系统键名配置说明

| 键名格式 | 说明 |
|---------|------|
| `defaultPeerId` | 系统默认节点标识 |
| `ed25519_privateKey` | 节点创建时生成的ED25519加密私钥 |
| `userBoxMaxKey_${publicKey}` | 用户离线消息同步标记（替换`${publicKey}`为实际用户公钥） |
| `ed25519_publicKey` | 最近登录用户的ED25519公钥 |
| `token_${publicKey}_${peerId}` | 用户节点访问凭证（需替换`${publicKey}`为用户公钥，`${peerId}`为目标节点ID） |
